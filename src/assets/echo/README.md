# Assets de l'Écho (le revenant)

L'**Écho** est le fantôme d'une de tes morts : il garde la salle où tu es
tombé·e (GDD §5.3). Par défaut son rendu est procédural (silhouette spectrale
violette) ; dépose ici deux feuilles pour lui donner ton art :

| Fichier (`.gif` ou `.png`) | Rôle |
|---|---|
| `echo-idle` | en sommeil sur son sigil, puis déplacement (boucle) |
| `echo-act` | action signature : télégraphe de tir / ruée (boucle) |

Mêmes règles que les ennemis (`../foes/README.md`) : frames **carrées**, fond
transparent, GIF prime sur PNG, les DEUX feuilles doivent être présentes pour
activer le skin. Idée de direction artistique : une version **spectrale et
blafarde du héros** (violet pâle `#b9a8ff`, contours d'encre, cœur lumineux) —
c'est « toi », en écho.
